Entrar

Use o usuário criado no Supabase → Authentication → Users.

Primeira vez?

  1. No Supabase: Authentication → Users → Add user → Create new user.
  2. Confirme que Email provider está habilitado em Authentication → Providers.
  3. Depois de entrar, o sistema tenta criar seu registro em public.usuarios (função “leitor”).

Obs: se o login der erro de chave/URL, confira js/supabaseClient.js.
